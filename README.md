---

## Getting Started
This project require NodeJS v12.

### Install
Clone this project
```bash
> git clone https://github.com/yaelahda/BOTWhatsapp_Pricelist
> cd BOTWhatsapp_Pricelist
```

Install the dependencies:
```bash
> npm install
```

Usage
```bash
> npm start
```
